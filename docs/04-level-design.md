# Divine Punishment — レベルデザイン仕様

作成日: 2026-07-03
ステータス: Draft v1
前提: [02-game-design.md](02-game-design.md) §8

## 1. レベルデザイン原則

1. **経路 3 系統ルール**: 標的 / 目標への到達経路を必ず 3 系統以上（地上 / 頭上 / 床下・水路）用意する
2. **観察ポイント**: 各区画の入口に、安全に敵配置を見渡せる場所（塀の上・茂み・軒下）を置く
3. **光は資源**: 消せる灯りと消せない月明かりを混在させ、闇を「作る」判断を要求する
4. **失敗しても続行**: 発見されても全滅リスクではなく、逃走 → 再潜伏経路（床下口・屋根）を各区画に置く
5. **音の地形**: きしむ板・砂利を「音のコスト地帯」として経路選択に組み込む

## 2. チュートリアル「山道の関所」

- 目的: 関所を突破し、密書を奪って脱出
- 敵: 4 体（巡回 2・立哨 2）。プレイ時間 5〜10 分
- 構造: 直列 5 区画。各区画で 1 メカニクスを強制学習

| 区画 | 教える内容 | 仕掛け |
|---|---|---|
| 1 山道 | 忍び足・可視度 HUD | 月明かりと木陰のコントラスト、背中を向けた立哨 1 |
| 2 崖沿い | 茂み・物陰・覗き見 | 巡回 1 体を茂みでやり過ごす |
| 3 番小屋前 | 小石の誘導・足音材質 | 砂利地帯、誘導しないと通れない立哨 |
| 4 番小屋 | 消灯・背後必殺・死体隠し | 灯篭 2、必殺対象 1、床下の隠し場所 |
| 5 関所門 | 経路選択（総合演習） | 地上（誘導 + 消灯）or 屋根（登攀）の 2 択 |

- リザルトあり（評価システムに触れさせる）。アートはグレーボックス + 簡易マテリアルで可

## 3. 本番ミッション「武家屋敷の粛清」

### 3.1 ミッション概要

| 項目 | 内容 |
|---|---|
| 目的 | 家老・毒山刑部（仮名）を暗殺し、屋敷から脱出する |
| 時間帯 | 夜半。月明かり + 灯篭・篝火 |
| 敵 | 10 体 + 標的 1（§3.4） |
| 想定プレイ時間 | 15〜30 分 |
| チェックポイント | 侵入成立時（塀内）・母屋到達時・暗殺成立時 |

### 3.2 マップ構造（4 層）

```
[外周] 正門(篝火・立哨2) / 東側 竹藪→塀 / 西側 水路→池
   │
[庭]   池・庭石・砂利(音地帯)・灯篭4・巡回2・提灯持ち1
   │
[母屋1階] 縁側→廊下(きしむ板)→座敷群→書院(標的)  巡回2・立哨2
[頭上層]  塀→屋根→屋根裏→梁(書院・座敷の上)
[床下層]  水路→床下口→床下(匍匐)→書院直下(隙間から視認可)
```

### 3.3 経路 3 系統

| 系統 | ルート | 特徴・要求スキル |
|---|---|---|
| A 地上 | 東竹藪 → 塀乗り越え → 庭 → 縁側 → 廊下 → 書院 | 灯り最多。消灯 + 誘導 + 提灯持ちの回避。忍具消費が激しいが観察はしやすい |
| B 頭上 | 正門脇 → 屋根 → 屋根裏 → 書院の梁 → 飛び降り必殺 | 移動は安全。ただし屋根への取り付き 2 箇所は立哨の視線が通る。梁からは護衛の位置次第で降りられない（観察と待ちが必要） |
| C 床下 | 西水路（潜行・息制限） → 床下口 → 床下 → 書院直下 → 引きずり込み必殺 | 最短。水中の息管理と床下のきしむ板。書院に標的がいる時間帯しか成立しない |

### 3.4 標的ルーチン（時間で回る、周期約 6 分）

```
書院で書見(2分) → 廊下→厠(1分, 護衛は廊下で待機) → 座敷で酒(2分, 護衛2) → 書院へ戻る(1分)
```

- 書院: 経路 B / C の必殺文脈が成立する唯一の場所（梁直下・床の隙間直上）
- 厠: 護衛と分離される唯一の時間（経路 A の背後必殺チャンス）
- 座敷: 護衛 2 で最難。ここで狙うのは上級者向け（時間短縮）
- **観察の報酬**: 遠くから 1 周期観察すれば、自分の経路に合う暗殺地点が判る

### 3.5 敵配置（10 体）

| 配置 | 数 | タイプ |
|---|---|---|
| 正門 | 2 | 立哨（篝火付き、消せない） |
| 庭 巡回 | 2 | 足軽（ルート交差あり） |
| 庭 | 1 | 提灯持ち（動的光源） |
| 縁側・廊下 | 2 | 立哨 1・巡回 1（きしむ板地帯を持ち場にする） |
| 座敷 | 1 | 休憩（時間で巡回に交代） |
| 護衛 | 2 | 標的ルーチンに随伴 |

エリア警戒レベル上昇時: 巡回が二人組化・提灯持ちが庭を追加巡回（実装は敵の `RoutineStop` 切替で表現）。

### 3.6 脱出

暗殺成立で目標が「脱出」に変わり、侵入経路 + 追加の脱出口（西水路の格子が内側から開く）が使える。警戒が上がっていると正門は封鎖される。

## 4. 制作チェックリスト（レベルごと）

- [ ] グレーボックスで経路 3 系統がそれぞれ未発見クリア可能（開発者テスト）
- [ ] 各区画に観察ポイントと再潜伏経路がある
- [ ] 消せる光源と消せない光源の比率がチューニング表と一致
- [ ] SearchPoint / RoutineStop / CheckpointArea 配置済み
- [ ] 基準時間（疾風評価）を実測で設定
- [ ] デバッグ可視化（視野・光量・音）で理不尽検知がないことを確認
