# Divine Punishment — ビジョン & コンセプト

作成日: 2026-07-03
ステータス: Draft v1

## 一行コンセプト

戦国の闇を舞台に、光と影を読み、影から影へ渡って標的に忍び寄り、一撃で任務を終えて消える cinematic stealth-puzzle ninja game。

『天誅4 (Tenchu: Shadow Assassins)』の精神的続編。公式 IP・キャラクター名・固有設定は一切使用しない（inspired by であり、続編を名乗らない）。

## 出典

本ビジョンは [tenchu4-concept-research.md](../tenchu4-concept-research.md) のコンセプト調査に基づく。

## デザインピラー（優先順位順）

| # | ピラー | 意味 | 実装への含意 |
|---|---|---|---|
| 1 | 発見されないことが主目的 | 敵を倒す数ではなく、気付かれない接近が成功条件 | 評価システムは未発見・痕跡なしを最上位に置く |
| 2 | 光と影は可変の盤面 | 灯りを消して闇を作れる。ステージはプレイヤーが変化させられる stealth puzzle | 光源は動的オブジェクト。可視度は光量から計算する |
| 3 | 接近の緊張 → 必殺の解放 | 標的に届くまでが緊張、一撃必殺の成功が報酬 | 必殺演出は短く濃く。接近中の UI/音響で緊張を演出 |
| 4 | 戦闘はリカバリ | 発見時の斬り合い・逃走は失敗からの立て直しであり、主目的ではない | 戦闘は簡素で不利。再潜伏（警戒解除）ルートを常に用意 |
| 5 | シネマティック和風暗殺劇 | 渋い時代劇の質感。Souls/Sekiro 風の派手なアクションに寄せない | トーン・アート・音響の統一。妖術なしの低ファンタジー |

## やらないこと（アンチゴール）

- ただの剣戟アクション / 敵を大量に倒す爽快アクション
- 忍者スキンのアリーナバトル
- ルートが一本道で「忍び寄り方を考える」余地がないレベル
- 公式 IP 名・キャラ名・設定の流用
- Wii モーション操作の再現（現代的なキーボード / ゲームパッド操作に置き換える）

## プレイヤーファンタジー

「闇に生きる忍として、汚れた標的に天誅を下す。誰にも見られず、誰にも知られず。」

## コアループ

```
隠れる → 観察する（視線・巡回・光源・地形を読む）
      → 経路を組み立てる（灯りを消す / 敵を誘導する / 死角を渡る）
      → 忍び寄る → 一撃で倒す → 痕跡を消す → 次の経路へ
```

失敗分岐: 発見 → 斬り合い or 逃走 → 再潜伏 → ループ復帰（理想プレイは発見ゼロ）。

## 成功の定義（Vertical Slice 完成条件）

1. コアループ（隠密・観察・接近・一撃必殺・痕跡処理）が 1 周まわり、緊張と解放が体感できる
2. 光・音・視線・警戒段階のルールがプレイヤーに読める（理不尽な発見がない）
3. 本番品質ステージ 1〜2 面（グレーボックス卒業、アート・音響込み）
4. ミッション評価（未発見 / 死体未発見 / 標的一撃 / 時間）が機能する
5. Steam 配信可能なビルドパイプラインが通る（配信自体はスコープ外）
