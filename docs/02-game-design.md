## 1. プレイヤー

### 1.1 スタンスと移動

| スタンス | 速度 (m/s) | 発生音半径 | 可視度補正 | 備考 |
|---|---|---|---|---|
| 忍び足（しゃがみ） | 1.5 | 1 m | ×0.6 | 既定の攻略スタンス |
| 歩き | 3.0 | 4 m | ×1.0 | |
| 走り | 6.0 | 12 m | ×1.3 | 緊急時・非警戒区間用 |
| 匍匐（床下） | 1.0 | 1 m | — | 床下専用 |
| 水中潜行 | 1.2 | 0 m | 水面波紋のみ | 息 20 秒、切れると強制浮上 + 音 |

床材質による足音倍率: 畳 ×0.5 / 木板 ×1.0 / きしむ板 ×2.0 / 砂利 ×1.5 / 浅水 ×1.8。

### 1.2 移動アクション

- **張り付き / 覗き見**: 壁・襖・柱に張り付き、角からカメラだけ覗ける（体は露出しない）
- **よじ登り**: マーカー付きの縁・塀・柱を登る。梁・屋根が頭上経路
- **床下潜入**: 床下口から匍匐経路へ。床の隙間から上の敵の足元が見える
- **隠れ場所**: 茂み・戸棚・天井裏。中にいる間は視覚検知対象外（ただし目前で入るのを見られたら無効）
- **飛び降り暗殺 / 床下からの引きずり込み**: 頭上・床下経路の出口報酬（§3）

### 1.3 HUD（最小主義）

- 可視度インジケータ: 自機足元のリング（暗いほど閉じる）
- 発音インジケータ: 音を発した瞬間だけ波紋表示
- 敵警戒: 敵頭上に発見メーター（不審=白 / 捜索=黄 / 戦闘=赤）
- 忍具スロット・残数、目標コンパスは方角のみ（正確なマーカーなし）

## 2. ステルスルール（盤面の物理）

### 2.1 可視度（Visibility）

プレイヤー可視度 `V ∈ [0,1]`:

```
V = clamp( 光量L × スタンス補正 × 移動補正 × 遮蔽補正 )
  光量L: 周囲の LightSource から距離減衰・遮蔽レイキャストで算出（完全な闇=0.05、直射=1.0）
  移動補正: 静止 0.8 / 移動中 1.0 / 走り 1.3
  遮蔽補正: 茂み等の SoftCover 内 ×0.3
```

敵の視覚検知速度は `V` に比例する（§4.2）。**V は HUD に常時表示され、プレイヤーは常に自分がどれだけ見えるかを知っている**（ルールの読める化、ピラー2）。

### 2.2 光源

| 光源 | 消し方 | 再点灯 |
|---|---|---|
| 灯篭 | 隣接インタラクト / 吹き矢 | 気付いた敵が約 60 秒後に点け直す |
| 篝火 | 水袋（VS では吹き矢で代替可否を検証） | 同上 |
| 提灯（敵携行） | 敵撃破でのみ消える | なし |

消灯は「異常」イベントを発し、視認した敵は不審状態になる（消しすぎのリスク）。

### 2.3 音

音イベント = (位置, 音量半径, 種別)。種別: 足音 / 着地 / 忍具 / 戸の開閉 / 戦闘音 / 悲鳴。
半径内の敵の聴覚に届き、遮蔽（壁）で半減する。悲鳴・戦闘音は警戒を即時「捜索」以上へ引き上げる。

## 3. 暗殺（必殺）

### 3.1 成立条件

以下すべてを満たすとき、必殺プロンプトが出る:

1. 敵の警戒段階が「戦闘」でない
2. 文脈が成立: 背後 1.5 m 以内 / 頭上（梁・屋根の直下 4 m 以内）/ 床下（床下口・隙間直上）/ 角（張り付き中、角の向こう 1.5 m）
3. 敵がこちらを視認中でない

成立時は入力 1 回で確定成功。**確率要素なし**（ルールの一貫性、FR-KIL-02）。

### 3.2 演出

1〜2 秒の短いカメラ演出 + 血しぶき最小限 + 消音気味の SE。緊張の解放を音響（無音→一拍→環境音復帰）で作る。連続使用でも冗長にならない長さを最優先。

### 3.3 死体処理

- 死体は担いで移動可能（移動速度 ×0.6、忍具使用不可）
- 隠し場所: 茂み / 床下 / 水中 / 戸棚
- 放置死体を敵が発見すると「捜索」へ即時遷移し、以後そのエリアは巡回が厳戒化（二人組化）
- リザルトの「死体未発見」評価に直結

## 4. 敵 AI

### 4.1 警戒段階（グローバルではなく敵個体ごと + エリア警戒レベル）

```
平常 Unaware ──(不審物/微検知)──▶ 不審 Suspicious ──(確信)──▶ 捜索 Searching ──(視認)──▶ 戦闘 Combat
   ▲                                  │(空振り)              │(見失い60s)             │(見失い)
   └──────────(帰投 Return)◀──────────┴───────────────────────┴──── 捜索 Searching ◀──┘
```

| 段階 | 挙動 | プレイヤーへの表示 |
|---|---|---|
| 平常 | ルーチン（巡回 / 立哨 / 休憩 / 雑談） | メーターなし |
| 不審 | その場で凝視 → 音源 / 異常点へ歩いて確認 | 白メーター |
| 捜索 | 走って周辺を捜索、隠れ場所の一部を覗く、仲間に伝播 | 黄メーター、BGM 不穏層 |
| 戦闘 | 攻撃・追跡・応援呼び | 赤、BGM 戦闘層 |
| 帰投 | 持ち場へ戻る（警戒残置: 一定時間、視覚検知が 1.5 倍速） | メーター減衰 |

エリア警戒レベル: 死体発見・戦闘発生で恒久的に 1 段上がり、巡回密度が上がる（失敗のコストを持続させる）。

### 4.2 視覚

- 視野: 水平 110°、有効距離 15 m（明所の立ち姿）
- 発見メーター蓄積速度 ∝ プレイヤー可視度 V × 距離減衰 × 中心視補正
- メーター満了で段階遷移（平常→不審は 0.5 秒相当 @V=1.0 至近、暗所遠距離ではほぼ蓄積しない）
- 遮蔽レイキャストで頭・胸・腰の 3 点判定

### 4.3 聴覚・異常検知

- 音イベント半径内で反応。足音 1 回 = 不審まで、悲鳴 / 戦闘音 = 捜索へ即時
- 異常オブジェクト: 消えた灯り / 死体 / 開いた戸 / 倒れた小物。視認で不審 or 捜索

### 4.4 敵タイプ（VS）

| タイプ | 役割 |
|---|---|
| 足軽（巡回） | 基本。巡回経路を持つ |
| 番士（立哨） | 定点。要所の門・廊下を塞ぐ。誘導が必要な壁役 |
| 提灯持ち | 移動光源。周囲の闇を消しながら歩く動的障害 |
| 標的（要人） | 専用ルーチン（書院で書見 → 廊下 → 厠 など時間で移動）。護衛 1〜2 名付き |

## 5. 忍具（VS は 3 種）

| 忍具 | 効果 | 所持数 | 設計意図 |
|---|---|---|---|
| 小石 | 着弾点に音イベント（半径 6 m） | 10 | 誘導の基本手段 |
| 吹き矢 | 光源消灯 / 敵 1 体を 20 秒昏倒（音で起きる） | 5 | 遠隔で盤面を変える |
| 煙玉 | 半径 5 m の煙、敵の視覚を 5 秒遮断 | 2 | 失敗時の離脱専用（攻撃には使えない） |

将来拡張: 手裏剣・まきびし・水袋・鉤縄はデータ駆動で追加可能な構造にする（03-technical-design.md §6）。

## 6. 戦闘（リカバリ）

- 操作: 斬撃（3 連まで）/ 受け流し（タイミング防御）/ 回避ステップ
- プレイヤー HP は敵の攻撃 3 発で死亡。敵 1 対 1 なら勝てるが、2 体以上では受け流し必須で現実的に不利
- 敵は応援を呼ぶため、**最適解は常に離脱**: 煙玉 → 遮蔽切り → 隠れ場所 → 敵が捜索 → 帰投
- 死亡時は直近チェックポイントからリトライ（ロード 3 秒以内目標）

## 7. ミッション評価

| 項目 | 判定 | 配点 |
|---|---|---|
| 影walker（未発見） | 一度も「戦闘」に入られていない | 40 |
| 痕跡なし（死体未発見） | 死体発見イベント 0 | 25 |
| 一撃（標的一撃） | 標的を必殺で仕留めた | 20 |
| 疾風（時間） | 基準時間内クリア | 15 |
| 減点 | 標的以外の殺害 1 体につき −3（上限 −15） | |

ランク: 90+ 皆伝 / 75+ 奥伝 / 55+ 中伝 / それ未満 初伝。ランクとともに各項目の達成 / 未達成を明示し、「次は何を目指すか」を提示する（リプレイ動機）。

## 8. ステージ（VS）

### 8.1 チュートリアル「山道の関所」

小規模（敵 4 体）。教える順: 忍び足と可視度 HUD → 茂みと物陰 → 音と誘導（小石） → 消灯 → 背後必殺 → 死体隠し → 脱出。各要素を 1 つずつ強制する直列構造だが、最後の関所だけ経路 2 択。

### 8.2 本番「武家屋敷の粛清」

- 標的: 屋敷の主（汚職の家老、夜半に書院で書見）
- 構造: 外周（庭・塀・池・砂利）/ 屋敷 1 階（畳・廊下・襖）/ 頭上（梁・屋根裏）/ 床下 + 水路
- 経路 3 系統: ①庭 → 縁側 → 廊下（地上・灯り多い・誘導と消灯が必要） ②塀 → 屋根 → 梁（頭上・移動は安全だが降下点が限られる） ③水路 → 床下 → 書院直下（最短だが息制限ときしむ板）
- 敵 10 体 + 標的。時間経過で標的が移動し、狙撃点が変わる（観察の報酬）
- 詳細は [04-level-design.md](04-level-design.md)

## 9. トーン・アート・音響方向

- 参照質感: 月明かり・行灯の暖色・藍の闇。彩度低め、コントラスト高め
- BGM: 平常時はほぼ環境音（虫・風・遠くの太鼓）。警戒段階で篠笛・鼓のレイヤーが重なる
- 妖術・超常演出なし。血表現は最小限（黒澤映画的な省略）
