### 0.3 共通ミッション構造

原則として各ミッションは `潜入 → （観察・攻略） → 主目標達成 → 脱出` + 任意の副目標 1 個（達成でリザルト加点 +5、修羅値を増やさない攻略の練習台になるものを選ぶ）。例外として M8 と M10 は副目標なし。

## M3 湊の蔵屋敷

| 項目 | 内容 |
|---|---|
| 目的 | 廻船問屋・戎屋徳兵衛を暗殺し、裏帳簿を奪って脱出 |
| 舞台 | 夜の湊。桟橋・帆船 1・蔵 3 棟・問屋母屋。水路が外周を巡る |
| 時間帯 / 天候 | 夜・晴れ。月明かり + 荷揚げ場の篝火 |
| ストーリー接続 | 06-narrative.md M3。内語「これも、務めだ。……務め、か」 |

- **層構造**: 水面（潜行）/ 桟橋・地上 / 蔵の屋根と渡り板 / 蔵内部（積み荷の迷路）
- **経路 3 系統**: A 桟橋正面（民間人の荷揚げ人足が多く、誘導主体・殺しの誘惑が最も強い経路）/ B 蔵屋根伝い（渡り板は音が大きい、見張り 2 の視線交差）/ C 水路 → 船底 → 母屋裏の船着き（息制限長め）
- **民間人 6**: 人足・遊女・番頭。敵ではないが玄羽を見ると悲鳴（半径 15 m の音イベント + 最寄り敵が捜索へ）。**殺害すると 1 人 −10 点 + 修羅値 +3**（通常敵 −3 の約 3 倍）
- **標的ルーチン（周期 5 分)**: 母屋 2 階の帳場（3 分・算盤の音が居場所を教える）→ 蔵 2 の検分（1 分・護衛 1）→ 桟橋で船頭と密談（1 分・周囲開けて狙撃困難、ただし水中から桟橋下に付ける）
- **副目標**: 抜け荷の阿片を海に捨てる（蔵 3・加点 +5）
- **敵構成**: 用心棒（足軽型）6・弓見張り 2（蔵屋根・新配置型: 高所静止、視界広いが背後が完全死角）
- **失敗回収**: 通報や発見で警戒が上がると標的は母屋に籠り護衛 2 が付く（難化するが攻略不能にはしない: 床下経路は残る）

## M4 雨の山寺

| 項目 | 内容 |
|---|---|
| 目的 | 刺客僧・鉄仙坊を暗殺し、匿われている綾辻の遺臣 2 名が斬られる前に救う |
| 舞台 | 石段・山門・本堂・鐘楼・宿坊・墓地。高低差の大きい縦積みマップ |
| 時間帯 / 天候 | 夜・**雨**（初出）: 足音 ×0.5・敵視界 −20%・灯篭は軒下のみ |
| ストーリー接続 | 「守るための殺し」の回。鉄仙坊の遺言「俺とお前の、何が違う」 |

- **時限プレッシャー（ソフト）**: 開始 12 分後（チューニング値）に僧兵が遺臣の座敷牢へ向かい始める。間に合わないと遺臣死亡＝副目標失敗（ミッションは続行可）。**ハード時限は採用しない**
- **経路 3 系統**: A 石段正面 → 山門（僧兵の定期読経で巡回が同期しており、鐘の音の間だけ全員が本堂を向く= リズム型パズル）/ B 崖 → 墓地 → 宿坊屋根 → 本堂の梁 / C 沢 → 水車小屋 → 床下 → 座敷牢直結（救出優先ルート）
- **鐘楼ギミック**: 鐘を鳴らすと全敵が本堂へ集まる（1 回限り・広域誘導。ただし Anomaly として警戒 +1）
- **標的**: 鉄仙坊は本堂で薙刀の稽古（固定）↔ 座敷牢の見回り（12 分後〜）。**戦闘になると全敵中最強**（受け流し必須・体力 2 倍）で「見つかったら負け」を体感させるボス級ガード
- **敵構成**: 僧兵 8（傘で視界の狭い下向き歩行 = 雨の日の敵挙動バリエーション）+ 鉄仙坊 + 座敷牢番 1
- **副目標**: 遺臣 2 名の救出（+5・H3 の台詞分岐に反映）

## M5 祭りの夜

| 項目 | 内容 |
|---|---|
| 目的 | 目付・黒沢帯刀を暗殺する。民間人被害ゼロが暗黙の課題 |
| 舞台 | 城下の祭り。参道・屋台通り・神社・桟敷・裏路地・屋根 |
| 時間帯 / 天候 | 夜・晴れ。提灯無数（消せない光源が主体の初のマップ） |
| ストーリー接続 | 中間の葛藤シーン（06-narrative.md §3.1）へ直結 |

- **群衆隠れ（初出）**: 雑踏 HideSpot（人混みに紛れる）。**走る・抜刀すると即座に悲鳴 → 周囲の敵へ伝播**。「光を消せないなら人に紛れろ」がテーマの逆転マップ
- **経路 3 系統**: A 雑踏（群衆隠れで参道を進む・最遅・最安全）/ B 屋根伝い（花火の明滅で可視度が周期変動: 花火が上がった瞬間 V×2、轟音で音は消える）/ C 裏路地 → 神社裏の井戸 → 用水路 → 桟敷の床下
- **花火システム**: 45 秒周期。打ち上げ 3 秒間: 屋根上の V×2 + 全音イベント無効（轟音マスク）。リズムを読む移動パズル
- **標的ルーチン（周期 6 分）**: 桟敷で観覧（3 分・両脇に護衛）→ 神社参拝（2 分・**拝殿で一人になる 30 秒**が最良の機会）→ 屋台通りを歩く（1 分・群衆の中、こちらも紛れて接近可）
- **敵構成**: 目付配下の同心 8・護衛 2。民間人 12
- **副目標**: 祭りの終わりまでに誰にも騒ぎを起こさせない（悲鳴イベント 0 で +5）

## M6 雪の峠

| 項目 | 内容 |
|---|---|
| 目的 | 峠を行く護送行列から密使・間宮小十郎を暗殺し、裏書状を奪う |
| 舞台 | 雪の街道・杉林・崖・炭焼き小屋・凍った沢。初の完全野外マップ |
| 時間帯 / 天候 | 薄暮 → 夜・**雪**（初出）: プレイヤーと敵の双方に足跡が残る |
| ストーリー接続 | 「何も知らぬ者を殺す」回。内語「ただの、御用。……俺と同じだ」 |

- **雪足跡**: プレイヤーの足跡は Anomaly（敵が視認すると不審 → 足跡を辿る）。降雪が 90 秒で足跡を消す。沢の岩伝い・倒木上は足跡が付かない（経路選択の資源）
- **行列 AI（初出）**: 駕籠 + 供侍 12 が街道を隊列移動。行列は 3 箇所の休止点（茶屋跡・峠の祠・炭焼き小屋）で停止し、その間だけ隊列が崩れて間宮が駕籠を出る
- **経路 3 系統**: A 街道に先回りして休止点に罠を張る（吹き矢で灯り持ちを眠らせ闇を作る）/ B 崖上から行列を俯瞰追跡し祠の単独参拝を狙う / C 凍った沢の下（氷上は音 ×2 だが足跡なし）から炭焼き小屋へ先回り
- **標的仕様**: 間宮は戦闘力ゼロ（抜刀できない若侍）。発見されても間宮は逃げるだけ——**殺しやすさそのものが情動設計**。必殺演出は本ミッションのみ簡略版（玄羽が一瞬ためらうモーション 0.5 秒追加・確定仕様）
- **敵構成**: 供侍 12（隊列時は相互視線でほぼ隙なし・休止点で 3〜4 人ずつに分散）
- **副目標**: 書状を誰にも知られず奪17う（行列が「暗殺に気づかないまま」峠を越えたら +5: 死体を沢へ隠す等）

## M7 忍びの里

| 項目 | 内容 |
|---|---|
| 目的 | 忍頭・鴉羽を暗殺し、里の宝蔵から連判状（真の宛先の証拠）を奪う |
| 舞台 | 玄羽の故郷の隠れ里。堀・跳ね橋・見張り櫓・修練場・宝蔵・頭屋敷 |
| 時間帯 / 天候 | 夜・晴れ |
| ストーリー接続 | 同門殺しの回。直後に千代が攫われる |

- **敵忍（初出・4 体）**: 視野 130°・梁 / 屋根を追跡可能（プレイヤー専用経路の聖域が消える）・Relight 即時・昏倒無効（吹き矢が効かない）・捜索時に忍具（撒菱）を使う。**ただし敵忍も闇では V ルールに従う = プレイヤーと同じ土俵**
- **罠**: 里には鳴子縄・撒菱が既設（Anomaly を出さず解除可能・解除は 3 秒硬直）
- **経路 3 系統**: A 正面の跳ね橋（夜間は上がっている → 櫓の巻き上げ機を操作。櫓は敵忍の定点）/ B 裏山の大杉 → 屋根へ跳ぶ（敵忍の巡回域・上級）/ C 堀 → 水門 → 修練場の床下 → 宝蔵裏
- **二重目標**: 鴉羽（頭屋敷・周期 4 分で修練場に指導に出る）と連判状（宝蔵・錠前は鍵持ちの敵忍から掏るか戸を破る=大音）。**順序自由**。鴉羽を先に殺すと里全体が恒久厳戒（宝蔵が難化）、連判状を先に取ると鴉羽が宝蔵へ確認に来る（待ち伏せ機会）——順序戦略の教科書マップ
- **敵構成**: 足軽型の下忍 6 + 敵忍 4 + 鴉羽（敵忍の強化個体）
- **副目標**: 同門の下忍を 1 人も殺さない（+5・H6 分岐に反映）

## M8 蔵屋敷の粛清

| 項目 | 内容 |
|---|---|
| 目的 | 側用人・梶原刑部少輔を暗殺する。全メカニクスの卒業試験 |
| 舞台 | 幕閣の蔵屋敷: 長屋門・白洲・母屋・蔵群・庭園・櫓。M2 の 2.5 倍規模 |
| 時間帯 / 天候 | 夜・**雨のち止む**（前半 15 分は雨補正、以降晴れ = 動的天候切替の唯一の使用箇所） |
| ストーリー接続 | 覚醒シーン（06-narrative.md §3.2）へ直結 |

- **厳戒状態から開始**: 玄羽の襲撃を予期しており、初期からエリア警戒 1（巡回二人組・SearchPoint 増）。「今までの当たり前が通じない」演出
- **経路 3 系統**: A 長屋門 → 白洲 → 母屋（正面・提灯持ち 3 の光の網）/ B 櫓 → 蔵の屋根 → 母屋の梁（敵忍 2 が屋根を巡回）/ C 庭園の池 → 雨水の溝 → 床下（雨が止むと溝の水位が下がり通行可になる = 天候連動経路）
- **標的ルーチン**: 梶原は書院に籠る（固定）。ただし 3 つの「おびき出し」が可能: ①蔵の火薬庫に細工（音で検分に出る）②偽の急使の文を門番に掴ませる（白洲へ出る）③何もしない（書院の梁からの一点狙い・護衛 2 が常時在室で最難）
- **敵構成**: 侍 8・提灯持ち 3・敵忍 2・護衛 2（計 14 + 梶原）
- **副目標**: なし（唯一の副目標なしミッション。「復讐の総仕上げ」に余事を置かない）

## M9 無血の城

| 項目 | 内容 |
|---|---|
| 目的 | 堀田家の城から千代を救出する。**殺害 = その場で任務失敗**（直前チェックポイントから再開） |
| 舞台 | 平城: 大手門・三の丸（番所）・二の丸（御殿）・天守下の奥御殿（千代）・水堀 |
| 時間帯 / 天候 | 夜・晴れ |
| ストーリー接続 | 誓いのシーン（§3.3）。刀と殺傷忍具を持たずに入る |

- **装備制限**: 刀なし（必殺・剣戟 UI 非表示）。忍具は小石・煙玉・**当身**（背後非殺傷テイクダウン・初出）・**縄**（昏倒 / 当身した敵を拘束: 覚醒しない・初出。ただし拘束体は死体同様 Anomaly）
- **発見時のルール変更**: 戦闘不能のため発見 = 即逃走。城兵は殺意でなく捕縛に来る（囲んで取り押さえ → 捕まると任務失敗）。煙玉と経路知識だけで切り抜ける
- **経路 3 系統**: A 大手門の物資荷車に隠れて搬入（荷改めの検分イベントを小石で逸らす）/ B 水堀 → 石垣の排水口 → 二の丸床下 / C 塀 → 櫓 → 御殿屋根（衛兵の提灯巡回が最も濃いが拘束で恒久的に「穴」を開けられる）
- **千代救出後**: 千代を連れた同行パート（走れない・隠れ場所は 2 人用のみ有効・千代は指示で待機 / 追従を切替）。脱出は排水口（B 経路の逆走）か、鐘楼の火災騒ぎ（副目標の布石）で開く大手門
- **敵構成**: 城兵 14（すべて非殺傷対象）
- **副目標**: 拘束 4 体以下でクリア（痕跡最小・+5） / 評価特例: 本ミッションの「一撃」項目は「当身成功数」で代替

## M10 天守の影（最終決戦）

| 項目 | 内容 |
|---|---|
| 目的 | 廃城の天守で宗玄を討つ |
| 舞台 | 五層の天守。層ごとに環境が異なる縦の決闘場 |
| 時間帯 / 天候 | 夜・雪（OP と対になる） |
| ストーリー接続 | 終幕（§3.4）→ エピローグ（§3.5） |

- **形式: 対忍決闘（逆ステルス）**。宗玄は「プレイヤーと同じルールで動く敵」: 隠れ、V に従い、音を聞き、忍具（撒菱・吹き矢・灯り消し）を使い、**こちらを探す**。HP 制ボス戦ではない
- **勝利条件**: 宗玄に必殺級の一撃を **3 回** 当てる。1 撃ごとに宗玄は煙玉で離脱し上の層へ（演出台詞 1 行ずつ・確定: ①「腕は上げたな」②「……何故、殺気がない」③〔終幕へ〕）
- **各層の設計**: 1 層: 燭台多数（消灯戦の基本）/ 2 層: 床が全面きしむ板（音の層・梁が安地）/ 3 層: 吹き抜けと梁のみ（垂直戦・宗玄も梁に乗る）/ 4 層: 障子の迷路（影絵: 光源を背にした影でお互いの位置が漏れる新ギミック）/ 最上層: 窓が全開で月光のみ・遮蔽は 4 本の柱だけ（隠れる場所がほぼない詰めの間合い。ここでは宗玄は隠れず、正面から「待つ」——決闘の最終形）
- **宗玄 AI 仕様**: 敵忍の強化。プレイヤーの忍具使用を学習（同じ誘導は 2 回目から無視・チューニング可）。プレイヤーが 5 分間攻撃機会を作れない場合、宗玄側から挑発の音を出す（膠着防止）
- **敗北条件**: 宗玄の必殺（背後を取られる）2 回で死亡。通常攻撃は受け流し可
- **評価特例**: 疾風なし。「影 walker」=宗玄以外の敵忍 4（1 層に配置）に見つからない、で代替
- **副目標**: なし（終幕の決闘に集中させるため）
