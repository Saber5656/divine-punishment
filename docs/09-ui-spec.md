# Divine Punishment — UI / 画面仕様書

作成日: 2026-07-03
ステータス: Draft v1

全画面共通: 和紙テクスチャの背景・墨色 (#1a1a1a 系) 基調・アクセントは朱 (#a03028)。フォントは明朝系（ライセンス確認の上、源ノ明朝を第一候補）。SE は筆 / 和紙の擦過音。解像度 1920×1080 基準、アンカーでスケール。

## 1. 画面フロー全体

```
起動 → タイトル → 章仕立てのミッション選択（巻物）→ Pre カットシーン → ミッション
     → Post カットシーン → リザルト → 隠れ家シーン → ミッション選択（次話解放）
Esc: ポーズ（再開/リトライ/設定/任務放棄）
```

## 2. タイトル画面

- 背景: 雪の夜の天守の遠景（エピローグ後は春の茶屋に差し替わる = クリア特典）
- メニュー: 「始める（続きから）」「最初から」「設定」「終わる」。縦書き・右端配置
- 初回起動時は「始める」→ 即 OP カットシーン → M1

## 3. ミッション選択（巻物 UI）

- 横長の巻物が広がり、第一夜〜第十夜が宿場のように並ぶ。未解放は霞で隠す
- 各話ノード: 題字・ベストランク印（皆伝=金 / 奥伝=朱 / 中伝=墨 / 初伝=灰）・達成フラグ 5 種の小紋（未発見 / 死体未発見 / 一撃 / 疾風 / 副目標）
- 選択パネル: 標的の人相書き（浮世絵）、お甲の口上（ミッション概要 3 行）、忍具持ち込み表示（変更不可・定義通り）
- クリア済みミッションは再挑戦可（ナラティブ変数へは**初回クリアのみ**反映。再挑戦はランク更新のみ）

## 4. HUD（ミッション中）

| 要素 | 位置 | 仕様 |
|---|---|---|
| 可視度リング | 画面中央下・自機足元感覚の円環 | V=0 で閉じた墨環 → V=1 で開いた白環。0.1 秒補間 |
| 音波紋 | リング外周 | 自分の音イベント発生時に 1 拍の波紋 |
| 敵メーター | 各敵頭上（3D→2D 投影、遮蔽時は画面端矢印） | 白（不審蓄積）→ 黄（捜索）→ 赤（戦闘）。塗り率 = メーター値 |
| 忍具 | 左下 | 現在スロット大 + 残数。ホイール/RB で巡回 |
| 息ゲージ | リング上 | 水中のみ表示 |
| 目標 | 右上・筆文字 1 行 | 「毒山刑部を討て」。更新時のみ 3 秒表示して消える |
| 方角 | 画面上端の細い帯 | 目標の方角に朱点（距離・マーカーなし） |
| 内語 | 画面下 1/3・縦書き 1 行 | 暗殺直後トリガー。4 秒でフェード。SE なし |
| インタラクト | 対象近傍 | 「消す」「担ぐ」「隠す」「開ける」「当身」— 動詞 1 語 + キー表示 |
| 必殺プロンプト | 対象敵の胸元 | 朱の刀紋。文脈（背後/頭上/床下/角）でアイコン差分 |

HP バーは置かない（被弾時に画面端が墨で染まる 3 段階表現）。M9 では刀紋 UI 全体が非表示、当身プロンプトに置換。

## 5. リザルト画面

- 巻物が縦に開き、上から: ランク大書（皆伝〜初伝・判子演出）→ 5 項目の達成表（○/×と点数）→ 標的以外の殺害数 → お甲の報告一言（修羅値分岐、06-narrative §1.3）→「次に目指すもの」（未達成項目のうち 1 つを提示）
- ボタン: 「先へ」（隠れ家シーンへ）/「もう一度」/「選択へ」

## 6. カットシーン / 隠れ家シーンプレイヤー

- 静止画をゆっくりパン / ズーム（Ken Burns、1 枚 8〜15 秒）
- テキスト: 画面下 25% の帯に横書き。ナレーション=白 / 台詞=話者名付き / 内語=斜体風の淡色
- 操作: 決定=次の行、長押し=早送り、Esc=スキップ確認。オート進行トグルあり
- 隠れ家シーンは背景 1 枚固定 + 会話のみ（カットシーンプレイヤーの縮退モードとして同一実装）

## 7. カットシーン画・画風ガイド（発注 / 生成用プロンプト規約）

- 様式: 浮世絵と挿絵の中間。太い輪郭線・平面的な色面・藍と墨が基調、朱は血ではなく灯りと紋にのみ使う
- 構図: 人物は影 / シルエット寄り。顔のアップは 06-narrative.md で指定された 2 箇所（M9 玄羽の目、M8 宗玄正面）のみ
- 禁止: 直接的な流血・現代的ディテール・写実 3DCG 調
- 全 70 枚は同一スタイルシード / 同一絵師で統一し、サンプル 3 枚のスタイル承認を経てから量産する（Issue 化済み）

## 8. 設定画面

| タブ | 項目 |
|---|---|
| 音 | 主 / BGM / SE 音量 |
| 操作 | 感度（X/Y）・反転・キーリマップ（全アクション、競合検知付き）・パッド振動 |
| 画 | プリセット（低/中/高）・フルスクリーン・垂直同期・明るさ（暗所基準画像付きキャリブレーション） |
| 遊び | 内語表示 on/off・HUD 縮小モード（可視度リングと忍具のみ） |

## 9. アクセシビリティ（VS 後・キャンペーンまでに必須の最小セット）

- 字幕サイズ 2 段階・テキスト送り速度
- 色弱対応: 敵メーターの白/黄/赤に形状差分（丸/三角/刀紋）を併用
- 明るさキャリブレーション（暗所ゲームの生命線）
