# 天誅4 コンセプト調査メモ

作成日: 2026-07-03

## 目的

Fable に「天誅4」の続編、または精神的続編を作らせる前提として、元作品がどのようなコンセプトのゲームだったかを整理する。

## 要約

天誅4は、戦国時代風の世界で忍者が闇に紛れ、敵に見つからず標的へ近づき、一撃必殺で任務を遂行するステルス忍者アクション。

ただし本作は従来作の単純な延長ではなく、2008年当時の公式発表で「今までの天誅とはまったく違うもの」「正統な天誅を押さえつつ、新しいもの」と説明されていた。

続編企画で継承すべき中核は、派手な剣戟ではなく、以下のプレイヤー体験。

- 発見されないこと自体が主目的
- 光、影、物陰、梁、水中、床下などを使って接近経路を組み立てる
- 標的に近づくまでが緊張、必殺が成功した瞬間に緊張が解放される
- 戦闘は失敗時のリカバリであり、基本報酬ではない
- 「どう近づくか」を考える stealth puzzle の性格が強い

## 天誅4らしさ

### 1. 戦闘よりも「接近」が主役

天誅4の快感は、敵を倒すボタンを押す瞬間だけではなく、その前の接近過程にある。

敵の視線、巡回、光源、隠れ場所、地形を読み、どのルートなら気付かれずに標的へ届くかを考える。

### 2. 光と影がゲームメカニクス

暗闇や物陰に隠れるだけでなく、篝火や灯篭の明かりを消して闇を作り出す。

つまりステージは単なる背景ではなく、プレイヤーが変化させられる stealth puzzle の盤面。

### 3. 一撃必殺のカタルシス

隠密状態からの必殺は、敵を一撃で倒す。

緊張が高いほど、成功した瞬間の解放感が強い。

### 4. 見つかったら負けに近い

戦闘はあるが、アクションゲームとして敵を斬り倒すことが主目的ではない。

発見された場合の戦闘や逃走は、あくまで失敗時のリカバリ。

### 5. 従来作からの再設計

公式発表では、天誅4は「今までの天誅とはまったく違うもの」を目指しつつ、「正統な天誅」を押さえる作品と説明されている。

そのため、続編を考えるなら、過去作の要素リストをなぞるより、天誅4が狙った「隠密体験の再発明」を継承するほうが重要。

## Fable への続編依頼で継承したい要素

| 継承したいもの | 現代化の方向 |
|---|---|
| 隠密、観察、接近、一撃必殺の流れ | 現代のキーボード / ゲームパッド / タッチ操作に合わせる |
| 影と光を使う stealth puzzle | 光源、音、気配、死角、隠れ場所を明確なルールにする |
| 力丸 / 彩女のような対照的プレイスタイル | 重い忍者と速い忍者、または道具型と身体能力型などに抽象化する |
| 戦闘より暗殺成功が快感 | 見つからない成功、死体を残さない成功、誰にも存在を知られない成功を評価する |
| シネマティックな和風忍者劇 | 派手な Souls / Sekiro 風アクションに寄せすぎない |

## 避けたい方向

- ただの剣戟アクションにする
- 敵を大量に倒す爽快アクションにする
- 忍者スキンのアリーナバトルにする
- ルートが一本道すぎて「忍び寄り方を考える」余地がない
- IP名、キャラ名、設定をそのまま使い、権利問題を起こす

## Fable に渡す短い企画文

```text
Tenchu 4 の精神的続編として、戦国の闇を舞台にした cinematic stealth-puzzle ninja game を作りたい。

プレイヤーは光、音、敵の視線、巡回、建築構造を読み、影から影へ移動して標的へ忍び寄る。
戦闘で勝つことより、見つからずに接近し、一撃で任務を終えて消えることが主な快感。

Wii motion controls は再現しなくてよい。
代わりに現代的な操作で、隠れる、観察する、忍び寄る、光源を消す、敵を誘導する、必殺を決める、というループを作る。

公式IPやキャラクター名は使わず、Tenchu 4 に inspired by な精神的続編として設計する。
```
