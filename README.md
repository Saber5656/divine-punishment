# divine-punishment

『天誅4 (Tenchu: Shadow Assassins)』に inspired された精神的続編 — 戦国の闇を舞台にした cinematic stealth-puzzle ninja game。

公式 IP・キャラクター名・設定は使用しない。

- エンジン: Godot 4 / PC (Steam 想定)
- 視点: 三人称 3D、箱庭ミッション構造
- 初期スコープ: Vertical Slice
